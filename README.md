# webpack-simple-react-typescript

Webpack 4 simple boilerplate for React 16 with Typescript 3 & SASS

![Webpack simple](webpack-simple.jpg)

# Includes
- React 16
- Typescript 3 loader
- Sass loader
- Bundling into HTML plugin
- Dev server with hot reloading

# Instructions

- Clone this repo
- `
npm install
npm start
`
